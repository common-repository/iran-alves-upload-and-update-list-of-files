Create and update list of files to be inserted in pages using shortcode.

== Description ==

Create and update list of files to be inserted in pages using shortcode. Great solution for important documents that must be updated constantly. Ideal for stock brokers, law firms, virtual product stores and many other purposes. A PRO version is being considered, so your feedback is welcome.


== Installation ==

1 - Upload the file "upload-and-update-list-of-files.zip" to the "/wp-content/plugins/" directory and unzip.
2 - In the "Plugins" menu, click "Enable" to enable it in the installation of WordPress.
3 - In the "Pages -> Iran Alves - Upload and Update List of Files" menu, select native WordPress pages to manage files.
4 - Insert the shortcode in current page editor, defining where file list will show.

== Frequently Asked Questions ==

= How does the plugin work? =
Upload a file or document, set it visible or not and also define the name of the file to be displayed for users to visit the page. It is necessary to include shortcode on the respective page to define the place where this list will be displayed.
= I liked the plugin and would like to contribute to the developer, how can I help? =
If this plugin helped you in any way evaluate the plugin in the repository of WordPress plugins, thank you immensely.
